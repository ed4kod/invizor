# Инструкция по использованию Invizor

## 🚀 Быстрый старт

1. **Запуск приложения:**
   ```bash
   python run.py
   ```

2. **Откройте браузер:** http://localhost:8000

3. **Вход в систему:**
   - Логин: `admin`
   - Пароль: `admin123`

## 📸 Загрузка изображений

### Поддерживаемые форматы:
- **JPG/JPEG** - стандартный формат для фотографий
- **PNG** - формат с поддержкой прозрачности
- **GIF** - анимированные изображения
- **WebP** - современный формат с хорошим сжатием

### Как загрузить изображение:

1. **При создании имущества:**
   - Нажмите "Добавить имущество"
   - Заполните название и описание
   - В поле "Изображение" выберите файл с компьютера
   - Нажмите "Добавить"

2. **При редактировании имущества:**
   - Откройте детали имущества
   - Нажмите "Редактировать"
   - Выберите новое изображение (или оставьте пустым для сохранения текущего)
   - Нажмите "Сохранить"

### Особенности:
- ✅ **Автоматическое изменение размера** - изображения оптимизируются
- ✅ **Уникальные имена файлов** - исключает конфликты
- ✅ **Безопасность** - проверка типов файлов
- ✅ **Предварительный просмотр** - видите текущее изображение при редактировании

## 🗂️ Управление счетами

### Создание счета:
1. На главной странице нажмите "+ Создать счет"
2. Введите название и описание
3. Нажмите "Создать"

### Просмотр имущества в счете:
1. Нажмите на карточку счета
2. Увидите список всего имущества в этом счете
3. Нажмите на карточку имущества для детального просмотра

## 🏷️ Управление имуществом

### Статусы имущества:
- 🟢 **Активное** - имущество в использовании
- ⚫ **Неактивное** - имущество не используется
- 🔴 **Повреждено** - имущество требует ремонта
- 🟡 **Утеряно** - имущество утеряно

### Редактирование:
- Измените название, описание, статус
- Замените изображение
- Все изменения сохраняются автоматически

## 🛠️ Техническая информация

### Размеры изображений:
- **Карточки в списке:** 400x300px
- **Детальный просмотр:** до 800x600px
- **Предварительный просмотр:** 128x128px

### Ограничения:
- **Максимальный размер файла:** 10MB
- **Поддерживаемые форматы:** JPG, PNG, GIF, WebP
- **Автоматическая оптимизация:** Да

### Безопасность:
- ✅ Проверка типов файлов
- ✅ Уникальные имена файлов
- ✅ Защита от переполнения
- ✅ Валидация на стороне сервера

## 🆘 Решение проблем

### Изображение не загружается:
1. Проверьте формат файла
2. Убедитесь, что размер не превышает 10MB
3. Попробуйте другой файл

### Ошибка при создании имущества:
1. Проверьте, что все обязательные поля заполнены
2. Убедитесь, что вы авторизованы
3. Перезагрузите страницу

### Проблемы с отображением:
1. Очистите кэш браузера
2. Проверьте подключение к интернету
3. Попробуйте другой браузер
